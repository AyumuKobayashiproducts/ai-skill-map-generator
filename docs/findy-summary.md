## AI Skill Map Generator（個人開発）- Findy 用サマリ

### プロジェクト概要

若手 Web エンジニア向けに、職務経歴とスキルセットを入力するだけで：

- スキルマップ（5カテゴリのレーダーチャート）
- 学習ロードマップ（30日 / 90日）
- 求人マッチング（マッチ度スコア・不足スキル）
- 転職準備スコア（0〜100点）
- ポートフォリオ整理（案件 TOP3 と Markdown 出力）

を自動生成する Next.js / Supabase / OpenAI ベースの Web アプリです。

```text
■ 担当・役割
個人開発として、要件定義〜設計〜実装〜テスト〜デプロイまでを一貫して担当しました。

■ 技術スタック
- フロントエンド: Next.js(App Router), TypeScript, Tailwind CSS, react-chartjs-2
- バックエンド: Next.js API Routes, OpenAI API(GPT-4o-mini), Zod
- データベース / 認証: Supabase(PostgreSQL, Auth, RLS)
- 品質: Vitest, Playwright, ESLint, Prettier, GitHub Actions

■ 工夫したポイント
- 転職活動の実務フロー（棚卸し→ロードマップ→求人マッチ→面接準備）を1本のユーザーストーリーとして設計し、画面遷移とコピーを揃えることで「3分で一周できる」体験を意識しました。
- スコア計算ロジックやAIプロンプト生成を lib/ 以下の純粋関数として切り出し、型定義とユニットテストで壊れにくい構造にしました。
- Supabase の RLS を前提にテーブル設計とクエリを整理し、認証まわりの誤操作でデータが漏れないよう配慮しました。
- GitHub Actions で type-check / lint / test / build を自動化し、PRベースで安全に機能追加できるようにしました。

■ 成果・学び
- AI を使った「診断系プロダクト」を、ビジュアル・スコア設計・UX まで含めて一気通貫で作りきった経験を得ました。
- TypeScript とテストを活用して、仕様変更に強いスコア計算・マッピングロジックを実現しました。
```
