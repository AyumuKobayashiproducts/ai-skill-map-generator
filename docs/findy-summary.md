## AI Skill Map Generator（個人開発）- Findy 用サマリ

> **AI と Supabase を使って、若手 Web エンジニアの転職準備を「スキル棚卸し → 学習ロードマップ → 求人マッチング → 面接練習」まで一気通貫で支援する Web アプリです。**

### プロジェクト概要

若手 Web エンジニア向けに、職務経歴とスキルセットを入力するだけで：

- スキルマップ（5カテゴリのレーダーチャート）
- 学習ロードマップ（30日 / 90日）
- 求人マッチング（マッチ度スコア・不足スキル）
- 転職準備スコア（0〜100点）
- ポートフォリオ整理（案件 TOP3 と Markdown 出力）

を自動生成する Next.js / Supabase / OpenAI ベースの Web アプリです。

### 💡 面接で語れる Before/After ストーリー

このプロジェクトは、当初「AI でスキルを診断するだけのツール」として構想しました。  
しかし、開発を進める中で「診断結果だけではユーザーは次の一歩を踏み出しにくい」という課題に直面しました。

そこで、単なる診断に留まらず、以下のような改善を加えました：

| Before | After |
|--------|-------|
| スキル診断結果のみ表示 | **学習ロードマップ**：具体的な学習計画を提示 |
| | **求人マッチング**：市場とのギャップを可視化 |
| | **面接練習**：アウトプットの場を提供 |
| | **ダッシュボード**：継続的な成長を可視化 |

この改善により、ユーザーが「診断 → 理解 → 行動 → 成長」というサイクルを回せるプロダクトへと進化させることができました。

この経験から、単に技術を実装するだけでなく、**ユーザーの課題に深く寄り添い、プロダクトの価値を最大化する視点**の重要性を学びました。
